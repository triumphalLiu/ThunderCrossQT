## 打飞机游戏——Qt版本
### 概述
编程语言：C++
IDE : Qt5

### 功能
请切换到英文输入
* W向上移动 A向左移动 S向下移动 D向右移动
* Esc退出游戏
* H射击
* P暂停游戏
* 右上角暂停/播放按钮可以打开会关闭游戏声音
